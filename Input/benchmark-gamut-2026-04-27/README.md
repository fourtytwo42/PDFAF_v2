# Benchmark Gamut 2026-04-27

Curated local benchmark pack covering protected fixtures, near-pass controls,
figure/alt ownership, structure and reading order, font/text extractability,
long mixed reports, table/link/annotation debt, manual/scanned policy debt, and
high-score controls.

PDF payloads in this folder are intentionally ignored by git. The tracked source
of truth is `benchmark-details.csv`.

The CSV includes local/source paths, problem family, benchmark role, expected
issue, before/after score and grade, and the run artifact used as the grade
source.
